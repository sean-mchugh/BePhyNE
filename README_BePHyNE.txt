BePhyNE data and scripts

For all materials go to https://github.com/sean-mchugh/BePhyNE this repository will be 
updated to include more materials and resources over time but currently includes

- the occurrence/pseudoabsence data for Eastern North American Plethodontids 
- BePhyNE package scripts (all currently as a single file to be executed) and a pipeline
 for setting up priors and running an MCMC using same settings as in our manuscript
